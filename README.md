# ansible_machine_rebuild
A repo for Ansible playbooks to rebuild machines

## Machines and Categories

### Mac Mini
+ ~mac-mini~
    + ~tranmission~
        + ~settings.json~
    + ~/etc/fstab~

### Dells
+ haho-services
    + docker_compose
    + /etc/fstab
+ haho-backup
    + /etc/fstab

### Raspberry Pi
+ pi4-4gb-dev
+ garage_pi
+ weather_pi
+ adsb_pi
